# Build Lab

AI-Augmented Product Explorations — a landing page showcasing 3 projects built with Claude Code, Figma MCP, and strategic design thinking.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS v4
- Framer Motion (stagger, hover, press animations)
- next-themes (system-aware dark/light)
- DM Serif Display + DM Sans (next/font/google)
- Lucide React (icons)

## Projects

- **Spendsmart** — Financial Advisor Powered by Claude ([GitHub](https://github.com/purnetm/Spendsmart))
- **Post Call Analysis** — No-Code Dashboard Builder ([GitHub](https://github.com/purnetm/PostCallAnalysis))
- **Calora** — E-commerce with AI Recommendations ([GitHub](https://github.com/purnetm/Calora))
